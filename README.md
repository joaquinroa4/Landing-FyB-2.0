# Landing_FyB

Landing page desarrollada con **HTML, CSS y JavaScript**.  
Creada para que el cliente pueda mostar sus obras terminadas
así como los servicios que brinda en todo momento
Diseñada para ser moderna, responsiva y fácil de personalizar.

## 🚀 Características
- Diseño responsivo adaptable a móviles, tablets y escritorio.
- Animaciones y efectos visuales con CSS y JavaScript.
- Secciones típicas de una landing page: encabezado, servicios, contacto.
- Código limpio y organizado para fácil mantenimiento.

## 📂 Estructura del proyecto
- /css --> Estilos principales
- /js --> Funciones y scripts
- /img --> Imagenes utilizadas 
index.html --> Página principal

## ⚙️ Instalación y uso
1. Clonar el repositorio:
   ```bash
   git clone https://github.com/joaquinroa4/Landing_FyB.git
2. Abrir el archivo "index.html" en el navegador
3. Personalizar estilos en /css y scripts en /js según se requiera

## 📦 Requisitos

1. Navegador moderno (Chrome, Edge, Firefox, Safari).
2. No requiere instalación de dependencias externas.
